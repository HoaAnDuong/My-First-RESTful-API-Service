SENDING REQUESTS

Did you write an guide just for Sending Requests?

Yes. Because I don't want to let you just explore it yourself.

In this website, you can get, change and delete data of a Store's Database via Sending Requests. There are 4 Request Methods corresponding to 4 Actions on Store's Database:

  • GET (SELECT): Get datas from a Table(or a Row)
  • POST (INSERT): Insert data into a Table
  • PUT(UPDATE): Update data on a Row
  • DELETE(DELETE): Delete data on a Row

If you don't know how to Send Requests, you can use Python's Module called 'requests' to do that.

Here the syntax of these HTTP Methods:

  • GET: requests.get(<link>)
  • POST: requests.post(<link>,json=<dict>)
  • PUT: request.put(<link>,json=<dict>)
  • DELETE: request.delete(<link>)
  • To get the HTTP code: requests.<HTTP Methods>
  • To get the returning json: requests.<HTTP Methods>.json()

Is that all I need to Sending Requests? No, You need know how my webservice behave to these Requests, because what the server will do on these Requests are decided by the Developers (which is me), not on the type of HTTP Methods. In the guide below, I will explain in Python code as well as explaination. What are you waiting for? Let's get started!

1. Customers

Customers API:  http://127.0.0.1:5000/customers

Customers API from CustomerID: http://127.0.0.1:5000/customers/<id> (will return HTTP code 404 if not found)


Here are HTTP Methods for Customers:

  • GET Customers (get Customers API): 

requests.get("http://127.0.0.1:5000/customers")

  • GET Customers by CustomerID (get Customers API by CustomerID): 

requests.get("http://127.0.0.1:5000/customers/<id>")

  • POST Customers (insert a new Customers): 

requests.post("http://127.0.0.1:5000/customers",json={

"CustomerID": "<CustomerID>",

"CustomerName": "<CustomerName>",

"ContactName": "<ContactName>",

"Address": "<Address>",

"City": "<City>",

"PostalCode": "<PostalCode>",

"Country": "<Country>"

})

  • PUT Customers by CustomerID (update on a Customers row by CustomerID):

r equests.put("http://127.0.0.1:5000/customers/<id>",json= {

"CustomerName": "<CustomerName>",

"ContactName": "<Contactname>",

"Address": "<Adress> ",

"City": "<City>",

"PostalCode": "<PostalCode>",

"Country": "<Country>"

})

  • DELETE Customers by CustomerID (set Status of a Customers row to 'unavailable')

requests.delete("http://127.0.0.1:5000/customers/<id>")
